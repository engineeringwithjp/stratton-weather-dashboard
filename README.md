# Stratton Weather Dashboard

A simple, clean weather dashboard built to display real-time conditions for Stratton Mountain.  
This project is part of my learning journey into web development, Git, and GitHub.

## Features
- Live weather data
- Clean UI
- Mobile-friendly layout
- Social preview images included

## Technologies Used
- HTML
- CSS
- JavaScript (optional if you add API calls later)

## How to View
Open `index.html` in your browser.
